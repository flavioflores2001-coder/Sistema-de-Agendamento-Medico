# Sistema de Agendamento Médico

## Descrição

Sistema web desenvolvido em PHP e MySQL para gerenciamento de consultas médicas.

O sistema possui autenticação por níveis de acesso e contempla as rotinas de uma clínica médica, como cadastro de pacientes, médicos, consultas, atendimentos, receitas e atestados.
